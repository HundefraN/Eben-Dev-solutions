# Eben Dev Solutions Website

This is the source code for the Eben Dev Solutions website.

## Files

*   **index.html**: Main HTML structure using semantic HTML5 tags.
*   **style.css**: Custom CSS3 styling with responsive design, animations, and dark/light mode support.
*   **script.js**: JavaScript for interactivity (menu toggle, theme switch, scroll animations).

## Features

*   **Responsive Design**: Looks great on mobile, tablet, and desktop.
*   **Dark/Light Mode**: User preference is saved and applied automatically.
*   **Animations**: Smooth transitions, floating elements, and scroll reveal effects.
*   **Modern UI**: Glassmorphism, gradients, and custom cursor for a premium feel.

## How to Run

Simply open `index.html` in your web browser. No build steps or servers required.
